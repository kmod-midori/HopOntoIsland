# 一键上岛

## 简介
识别屏幕上的取餐号/排位号/取件码/车牌号等，显示在Android的实时通知中。

## 使用方法
1.Gemini模型
前往[Google AI Studio](https://aistudio.google.com/)
获取API Key后填入应用中，无需填入URL，模型默认填入"gemini-2.5-flash"，点击保存即可
